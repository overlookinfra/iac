DevX Team Update

It’s been a busy week on the DevX team! We’re delighted to announce the release of PDK version 2.1.1. This release resolves an issue in which child processes were unable to spawn on some Windows systems. Check out the release notes for a full list of enhancements and fixes that have gone in to 2.1.1. We also have a new release of the PCT tool. Version 0.2.0 can be installed over the top of 0.1.0 using these installation instructions. Check out the the 0.2.0 release page for full details of what’s included.
